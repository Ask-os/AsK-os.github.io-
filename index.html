Conocimientos

DJANGO

Utilizo Django como framework de desarrollo web debido a su robustez y enfoque en la seguridad desde el inicio. Su arquitectura modular me permite desarrollar aplicaciones escalables y seguras rápidamente, lo que optimiza los tiempos de desarrollo.

BASE DE DATOS

Trabajo con bases de datos porque me permiten almacenar, organizar y consultar grandes volúmenes de datos de forma eficiente. Las bases de datos relacionales, como MySQL, son fundamentales para mis proyectos ya que proporcionan la estructura necesaria para manejar la información de manera fiable y escalable.

SHELLS

Uso Shell (bash/zsh) para administrar entornos Unix/Linux y automatizar tareas repetitivas. Estas terminales me permiten gestionar procesos, realizar scripting avanzado y optimizar la administración del sistema de manera eficiente. Gracias a su flexibilidad, puedo personalizar y automatizar flujos de trabajo, lo que me ahorra tiempo y mejora la productividad en mis proyectos.
